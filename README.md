# Claude Prompts

A collection of practical prompts for getting more out of Claude Code.
Paste the contents of each prompt file into Claude to use immediately.

## List

| Prompt | Description |
|---|---|
| [meeting-recorder](./meeting-recorder/README.md) | Record meetings, extract transcripts, and translate |
| [session-manager](./session-manager/README.md) | Search, resume, and import Claude sessions across projects |
| [profile-switcher](./profile-switcher/README.md) | Switch Claude API endpoint profiles per environment |
